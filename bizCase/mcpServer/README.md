# MCP Demo

这是一个基于 mark3labs/mcp-go 实现的 MCP 示例项目。

## 功能特性

1. 计算器工具
   - 支持加、减、乘、除四则运算
   - 包含参数验证和错误处理

2. 时间工具
   - 获取当前时间
   - 支持自定义时间格式

## 运行方式

```bash
go run main.go
